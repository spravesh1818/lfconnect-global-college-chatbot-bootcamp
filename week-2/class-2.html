Notebook · Section 26

LCEL — The Pipe Operator

from langchain_core.output_parsers import StrOutputParser

chain = zero_shot | llm | StrOutputParser()
result = chain.invoke({"topic": "photosynthesis", "duration": "45-minute"})
print(type(result))   # str, not AIMessage
print(result[:200])

| composes runnables. StrOutputParser converts AIMessage → str so the result can feed into the next step.

Notebook · Section 27

Sequential Chains — Two-Step Workflow

summary_prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a concise Biology tutor."),
    ("human", "Summarize {topic} in 3 bullet points."),
])
quiz_prompt = ChatPromptTemplate.from_messages([
    ("system", "You are an exam question writer."),
    ("human", "Based on this summary:\n{summary}\nWrite 3 MCQ questions."),
])

summary_chain = summary_prompt | llm | StrOutputParser()
quiz_chain    = quiz_prompt    | llm | StrOutputParser()

summary   = summary_chain.invoke({"topic": "photosynthesis"})
questions = quiz_chain.invoke({"summary": summary})
print(questions)

Tools & Agents

Give the LLM the ability to take real-world actions

What Are Tools?

  • Tool — a Python function the LLM can call (search, calculate, look up data)
  • Agent — LLM + tools + a reasoning loop
  • ReAct pattern — Reason → Act (call tool) → Observe result → Repeat
User input→ LLM reasons→ Calls tool→ Observes→ Final answer

Notebook · Section 29

Custom Python Tool

from langchain.tools import tool

@tool
def calculate(expression: str) -> str:
    """Evaluate a math expression. Use for arithmetic calculations."""
    allowed = set("0123456789+-*/()., ")
    if not set(expression).issubset(allowed):
        return "Error: unsafe expression"
    return str(eval(expression))

print(calculate.invoke("12 * 6"))   # "72"

The docstring is the tool description — the LLM reads it to decide when to use this tool.

Notebook · Section 31

Build the Tool Agent

from langchain.agents import create_tool_calling_agent, AgentExecutor
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

agent_prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a research assistant. Use tools when needed."),
    ("human", "{input}"),
    MessagesPlaceholder("agent_scratchpad"),
])

tools = [search, calculate]
agent    = create_tool_calling_agent(llm, tools, agent_prompt)
executor = AgentExecutor(agent=agent, tools=tools, verbose=True, max_iterations=5)

verbose=True shows every tool call + observation — the ReAct loop live.
