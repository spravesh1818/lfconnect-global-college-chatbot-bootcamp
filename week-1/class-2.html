Notebook · Section 8

Minimal API Call

from groq import Groq

client = Groq(api_key=os.environ["GROQ_API_KEY"])
MODEL = "llama-3.3-70b-versatile"

response = client.chat.completions.create(
    model=MODEL,
    messages=[{"role": "user", "content": "What is a chatbot?"}],
)
print(response.choices[0].message.content)

Verify the latest model name on Groq docs — model IDs change over time.

Request Anatomy

FieldPurpose

messagesConversation history (roles + content)
temperatureCreativity (0 = focused, 1+ = varied)
max_tokensLimit on response length

])

])

The model did not get smarter — the instruction changed.

Notebook · Section 9b

Assistant Role = Memory

messages = [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "My favorite color is blue."},
]
first_reply = chat(messages)
messages.append({"role": "assistant", "content": first_reply})
messages.append({"role": "user", "content": "What is my favorite color?"})
print(chat(messages))

Without the assistant turn, the model would not remember what you said earlier.

Notebook · Section 9c

The chat(messages) Pattern

def chat(messages: list) -> str:
    response = client.chat.completions.create(
        model=MODEL,
        messages=messages,
        temperature=0.7,
    )
    return response.choices[0].message.content

def ask(question: str) -> str:
    return chat([{"role": "user", "content": question}])

Rate Limits & Latency

Rate limits

  • Requests per minute (RPM)
  • Tokens per minute (TPM)
  • HTTP 429 when exceeded

Latency

  • Time to first token
  • Total generation time
  • Larger models = slower, often smarter

Add time.sleep() between calls if you hit 429 errors.

Notebook · Section 15

Robust API Call Pattern

def ask_safe(question: str) -> str:
    try:
        response = client.chat.completions.create(
            model=MODEL,
            messages=[{"role": "user", "content": question}],
        )
        return response.choices[0].message.content
    except Exception as e:
        return f"Sorry, something went wrong: {e}"
