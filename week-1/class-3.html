What is a System Prompt?

The system message sets behavior before the user speaks — tone, role, constraints, and format.

  • Not shown to end users (in most UIs)
  • Steers every reply consistently
  • Week 2 goes deeper on prompt engineering

Personalized Chatbot — Code

from groq import Groq
import os

client = Groq(api_key=os.environ["GROQ_API_KEY"])
MODEL = "llama-3.3-70b-versatile"

SYSTEM_PROMPT = """You are a friendly programming tutor.
Explain concepts in simple steps."""

def chat(user_message: str) -> str:
    response = client.chat.completions.create(
        model=MODEL,
        messages=[
            {"role": "system", "content": SYSTEM_PROMPT},
            {"role": "user", "content": user_message},
        ],
        temperature=0.7,
    )
    return response.choices[0].message.content

Multi-Turn Conversations (Stretch)

Pass prior assistant messages so the bot remembers context:

history = [
    {"role": "system", "content": SYSTEM_PROMPT},
]

def chat_turn(user_message: str) -> str:
    history.append({"role": "user", "content": user_message})
    response = client.chat.completions.create(
        model=MODEL, messages=history
    )
    reply = response.choices[0].message.content
    history.append({"role": "assistant", "content": reply})
    return reply

Debugging Checklist

  • Is GROQ_API_KEY set? (echo $GROQ_API_KEY)
  • Correct model name from Groq docs?
  • Internet / VPN / firewall blocking API?
  • 429 rate limit — slow down or wait
  • Empty user input handled in your loop?

Common Issues (Q&A)

SymptomSolution
KeyError: GROQ_API_KEYExport env var or use python-dotenv
Bot ignores personalityPut instructions in system, not user
Replies too longAdd "Keep answers under N sentences" to system prompt
Off-topic answersTighten system prompt; lower temperature
Colab secretsUse Colab "Secrets" for API key storage
